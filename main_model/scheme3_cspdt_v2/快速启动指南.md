## 🚀 快速启动

### Stage 1: 冷启动训练 (200k steps)

```bash
cd /home/wangmeiyi/AuctionNet/medical/重构之后的代码/优化主要模型/scheme3_cspdt_v2

# 启动 Stage1 训练
python train_stage1.py --logdir ./checkpoints/stage1

# 或者使用 nohup 后台运行
nohup python train_stage1.py --logdir ./checkpoints/stage1 > logs/stage1.log 2>&1 &
```

**预计时间**: 约 10-20 小时 (取决于 GPU)

**检查点保存**: 每 30k steps 保存一次，位于 `checkpoints/stage1/step_*/`

### Stage 2: Self-Play 微调 (50 epochs)

```bash
# 等 Stage1 完成后，使用最终检查点启动 Stage2
python train_stage2.py \
  --logdir ./checkpoints/stage2 \
  --policy_ckpt ./checkpoints/stage1/step_200000/policy.pt \
  --world_model_ckpt ./checkpoints/stage1/step_200000/world_model.pt

# 或者后台运行
nohup python train_stage2.py \
  --logdir ./checkpoints/stage2 \
  --policy_ckpt ./checkpoints/stage1/step_200000/policy.pt \
  --world_model_ckpt ./checkpoints/stage1/step_200000/world_model.pt \
  > logs/stage2.log 2>&1 &
```

**预计时间**: 约 5-10 小时

## 💡 优化建议

1. **监控 sigma_mean**: 应该在 0.3-0.8 之间，如果过高说明模型不确定性太大
2. **监控 confidence**: 应该 > 0.5，如果过低说明很多候选动作被拒绝
3. **监控 bootstrap_pct**: 应该逐渐增加，表示模型越来越自信
